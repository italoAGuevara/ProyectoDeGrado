Cómo funcionan
  • Pre-copiado: se ejecutan antes de iniciar la copia (ej: comprimir, validar).
  • Post-copiado: se ejecutan después de terminar la copia (ej: notificar, limpiar).
  • El orden define la secuencia cuando hay varios del mismo tipo (menor = primero).
  • Solo se permiten scripts .ps1 (PowerShell), .bat (Batch) y .js (Node.js).
